Stardew Valley Fishing Bot


Welcome to the SVFB project, where people like me (who can't fish for shit on this game) can now have a chance at it!
The project is still not complete, I now need help collecting data from the game to finish the machine learning process. Read below how to help!

Download! »

How to help (please do)

If you can, download the program on the download button above, click start and play the game as normal! For it to work you need to use the C button to do all the fishing. The program does NOT work if you fish with left-clicks. (You will be able to change this on the future!)

Read more on reddit »

How it works

As soon as you click on start, the program will look on your screen for the fishing region, then, when the fishing mini-game starts, the program will record the fish's position on the screen, saving it together with you "C" clicks.
It stops recording as soon as the mini-game closes, and should automaticly send the data to this website if you allow it. Also, you can see how many frames you contributed in total on the top of this page! (with one super cool graphs that was hard as fuck to do!)
