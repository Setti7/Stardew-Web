How to help (please do)

If you can, download the program on the download button above, click start and play the game as normal! For it to work you need to use any keyboard button to fish (you just need to set it, the default is C). The program does NOT work if you fish with left-clicks.

Read more on reddit »

How it works

As soon as you click on start, the program will look on your screen for the fishing region, then, when the fishing mini-game starts, the program will record the fish's position on the screen, saving it together with you "C" clicks (or the any other keyboard button you use).
It stops recording as soon as the mini-game closes, and should automaticly send the data to this website.
